1002. Different GCD Subarray Query

题目链接

题目大意

给定一个长度为\(n\)的序列，有\(q\)次询问，每次询问区间\([l,r]\)内的连续子序列的\(gcd\)有多少个不同的值？

思路 - 树状数组 && 离线

想起多校有道类似的题，在\(O(nlog(max{a[i]}))\)内也可以用\(RMQ\) + 二分处理出以\(r\)为右端点的所有的\(gcd\)值。但是本题的题解处理方法太神了，完全想不到这个性质。
注意固定右端点\(r\)后，出现\(gcd\)值的区间\([l,r]\)是最小的区间，即\(l\)最大，这样可以保证不会漏算\(gcd\)值在某些区间出现的情况。
然后离线处理所有询问，将询问按照区间右端点升序排序，用树状数组维护每个\(gcd\)值在询问区间右端点前出现的最右边的位置即可。

1006. Football Games

题目链接

题目大意

有\(m\)个小组，每个小组有\(n\)个队，任意两个队之间都会有一场比赛。胜者得\(2\)分，负者得\(0\)分，平局各得\(1\)分，给定所有队的分数，判断是否合法？

思路 - 图论

完全想不到这是一个图论题。
只看到有Landau’s Theorem “Tournament (graph theory)”)的公式，并没有找到证明。。。
大概可以这样想？分数按照升序排序后，前\(i\)个人之间的比赛一定会产生\(i*(i-1)\)分，所以分数和不够的一定不合法。

1008. Function

题目链接

题目大意

给定一个数组\(a[1],a[2],…,a[n]\)，有\(m\)次询问\([l,r]\)，表示询问\(a[l] mod a[l+1] mod … mod a[r])的值？

思路 - RMQ && 二分

比赛时已经知道用RMQ + 二分了，但是一直\(TLE\)（其实是\(RE)\)）到结束，这个评测姬差评！
首先知道对于任意两个正整数\(a,b\)，有\(a\ mod\ b <= \frac {a} {2}\)。
所以\(a[l]\)最多会进行\(log(a[l])\)次模运算。而有效的模运算是模小于等于当前数，所以可以先用\(RMQ\)维护区间最小值，然后二分即可，时间复杂度\(O(mlognlog(max{a[i]}))\)。

1009. Sparse Graph

题目链接

题目大意

给定一个无向图，求其补图上\(S\)点到其他所有点的最短路？

思路 - BFS

今天一看是一道非常简单的水题，然而比赛时写这题的人一直处于即将\(AC\)的状态就没管了，早知道就看一下这题了。

由于边权都是\(1\)，所以直接\(BFS\)即可，但是补图边异常多，肯定存不下，也不能每次遍历所有点。然后就想到已经遍历过的点不会再次遍历，所以初始将所有的点放入集合中，对于当前点\(u\)，遍历还剩下的点\(v\)，若\((u,v)\)在原图上则不进行操作，否则更新最短路，并将\(v\)从集合中删除。

看了非官方题解后发现，用链表维护未结点也可以。

1010. Weak Pair

题目链接

题目大意

给定一棵有根树，每个点有一个权值\(a[i]\)，求有多少对点满足\(u\)是\(v\)的祖先结点（\(u!=v\)），且\(a[u] * a[v] <= k\)？

思路 - 树状数组

很容易就能想到\(dfs\)时维护一个有序的序列\(a[u]\)，进入\(dfs\)时更新答案后插入，出\(dfs\)时删除，则序列中的值都是当前结点祖先结点的，所以小于等于当前结点的\(k / a[u]\)的个数，就是满足题意的对数。但是必须在\(logn\)内维护这个序列，第一反应只想到\(splay\)，但估计常数太大会超时，就又去想别的方法，最后想到离散化后用树状数组弄一下就行了，比赛时\(1.9s\)险过。
结束后看了非官方题解，发现\(treap\)这个数据结构可以很方便的实现上述功能。

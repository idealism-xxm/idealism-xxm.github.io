HDU-1517 A Multiplication Game（巴什博奕变形||PN分析）

2016-07-21 • 分类: ACM, 博弈论 • 标签: ACM, HDU, PN分析, 博弈论, 巴什博奕

题目链接

HDU-1517 A Multiplication Game

题目大意

给定一个数\(n\)，初始\(p=1\)，每次可以给\(p\)乘上\(2~9\)，最先使\(p>=n\)的人获胜，求谁必胜？

Sample Input

162
17
34012226

Sample Output

Stan wins.
Ollie wins.
Stan wins.

思路

巴什博奕变形就又不会了.
只能想到刚开始的必胜态\([2,9]\)和必败态\([10,18]\)，后面就有点不好想了，就先用\(PN分析\)\(AC\)了。
后来想到先手掌握控制权，所以必胜态\([19,162]\)可以通过自己选数避免对方到达。
例如：\(n=19\)时，先手为了避免后手进入必胜态，所以他可以选\([2,ceil(19/9)-1]\)，然后后手不管怎么选，都不会胜利，而且先手下一次选必定获胜。

又看到\(PN分析\)的思路：
设\([n,+∞]\)为必败态
通过\(PN分析\)可知\([ceil(n/9),n-1]\)为必胜态，
接着可以推出\([ceil((n/9)/2),[ceil(n/9)-1]\)为必败态
…
最终可以推得\(1\)是必胜点还是必败点。

必胜点和必败点的性质：
1、所有终结点是 必败点P。
2、从任何 必胜点N 操作，至少有一种方式可以进入 必败点P。
3、无论如何操作，必败点P 都只能进入 必胜点N。

\(PN分析\)对于这种状态很少的博弈题还是很有效的。

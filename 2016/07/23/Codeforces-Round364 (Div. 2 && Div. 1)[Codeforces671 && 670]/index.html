671 - C - They Are Everywhere

题目大意

给定一个字符串，求长度最小的区间，使得这个区间包含所有出现的字符至少一次。

思路 - 尺取法

维护一个区间\([l,r]\)并统计区间内所有字符出现的次数，以及多少不同的字符数，每次\(r+1\)，若\(cnt[s[l]]>1\)，则\(l+1\)直至\(cnt[s[l]]==1\)。取不同的字符数等于字符串所有不同的字符时的区间长度的最小值。

670 - A - As Fast As Possible

题目大意

有\(n\)个人要走完\(l\)的距离，人的速度为\(v_1\)，现有一辆速度为\(v_2\)并且最多能载\(k\)人的车，每个人最多乘车一次，求所有人均走完\(l\)时的最短时间？

思路 - 数学

比赛时陷进去了，认为车一定会开到终点，然后怎么都跑不出第二组样例，第二天才想到车不一定开到终点，时间最短的情况下一定是所有人同时抵达终点，并且此时所有人坐车距离相等。
设坐车距离为\(s\)，车开出去再回来接下一批人用时为\(t\)，返回时用时为\(t^` \)总共有\(cnt=ceil(n/k)\)批人坐车，最后一批人直接坐到终点，设\(sta\)表示每次人上车的位置。

则：
\(t=\frac {2*s} { v_1 + v_2 } \)
\(t^=t-s/ {v_2} \\) 下次上车位置：\\(sta+v_1 \* t=sta+s-v_2 \* t^ \)
总共用时：\(t*(cnt-1)+ \frac {s} { v_2 } = \frac {s} { v_2 } +\frac {l-s} { v_1 } \)
解得：\(s=\frac { ( v_1 + v_2 ) * l } { v_1 + v_2 + 2*(cnt-1)* v_1 } \)
则总共用时：\frac {s} { v_2 } +\frac {l-s} { v_1 } \)

觉得列方程比较麻烦也可以二分\(s\)。

670 - B - Connecting Universities

题目大意

有一棵\(n\)个结点的树，选定树上的\(2*k\)结点，分成\(k\)对，使得这\(k\)对的最短路径和最大，求这个最大值？

思路 - 树形DP

假设\(1\)为原树的树根，\(cnt[u]\)表示以\(u\)为根的子树中选定结点的个数，每条边\( (u,v) \)对答案的贡献就是\(min(cnt[v],2*k-cnt[v]\)。

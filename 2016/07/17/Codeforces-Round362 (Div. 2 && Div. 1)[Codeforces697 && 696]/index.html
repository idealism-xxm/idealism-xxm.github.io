696 - A - Lorenzo Von Matterhorn

题目大意

给定一个二叉树，结点\(i\)的子节点分别为\(2*i和2*i+1\)，有两种操作：
①给\(u,v\)的最短路上每条边都加上\(w\)
②查询\(u,v\)之间的最短路

思路 - 模拟

由于给的是一个满二叉树，而且结点编号十分规律，所以可以直接不停找两个结点的父亲结点，并进行相应处理（①该边加上\(w\)，②答案加上该边权值），直到两个结点编号相同。

696 - B - Puzzles

题目大意

给定一个有根树，每次等可能地随机遍历当前结点的子节点，求每个节点访问时间的期望？

思路Ⅰ - 数学

对于每个结点u，可以将其他结点分成三类：
\(①：u的祖先结点，共depth[u]-1个，\)\(他们一定会在u之前被访问，所以他们对期望的贡献为1\)
\(②：u的子树上的结点，共cnt[u]-1个\)\(他们一定会在u之后被访问，所以他们对期望的贡献为0\)
\(③：既不是u的祖先结点，又不是u的子树上的结点，共n-cnt[u]-depth[u]\)\(他们中任意一个在u结点的前和后的概率均是0.5，\)\(即他们对期望的贡献为0.5\)
则：\(dp[u]=[n-(cnt[u]-1)-(depth[u]-1)-1]*0.5+(depth[u]-1)+1\)
\(=(n-cnt[u]-depth[u]+1)*0.5+depth[u]\)
\(=(n-cnt[u]+depth[u]+1)*0.5\)

代码

思路Ⅱ - 树形DP

对于每个结点u，可以将\(u的父亲结点pa[u]结点\)的子树上的其他结点分成三类：
\(①：u的父亲结点pa[u]，他的期望dp[pa[u]]对期望的贡献为1\)
\(②：u的子树上的结点，他们一定会在u之后被访问，所以他们对期望的贡献为0\)
\(③：u的兄弟结点，\)\(他们中任意一个在u结点的前和后的概率均是0.5，\)\(即他们对期望的贡献为0.5\)
则：\(dp[u]=dp[pa[u]]+(cnt[pa[u]]-cnt[u]-1)*0.5+1\)

696 - C - PLEASE

题目大意

给定三个杯子，初始把钥匙放在中间的杯子下，每次将中间的杯子随机与两边的任意一个杯子互换，问\(n\)次之后，中间的杯子下有钥匙的概率？

思路 - 数学

设\(dp[i]\)表示第i次后，中间的杯子下有钥匙的概率，则初始\(dp[1]=0\)
状态转移方程为：\(dp[i]= \frac { (1-dp[i-1]) } { 2 }\)
用数学解法可通过数列的递推公式解出数列通项公式：\(dp[i]={ {1- { ( - \frac {1} {2} ) }^{n-1} } \over {3} }\)
\(= \frac { { { (-2) }^{n-1} - 1 } \over {3} } { {(-2)}^{n-1} }\)
①当\(n\)为奇数时：\(dp[i]=\frac { { 2^{n-1} - 1 } \over {3} } { 2^{n-1} }\)，即：\(p={ { {2}^{n-1} - 1 } \over {3} }\)，\(q=2^{n-1}\)
②当\(n\)为偶数时：\(dp[i]=\frac { { 2^{n-1} + 1 } \over {3} } { 2^{n-1} }\)，即：\(p={ { {2}^{n-1} + 1 } \over {3} }\)，\(q=2^{n-1}\)

且①,②中的\(p,q\)均是互质的。

687 - D - Dividing Kingdom II

题目大意

有一个无向图，\(n\)个点，\(m\)条边，\(q\)次询问，每次询问边号为\([l,r]\)的边组成的图，将这个图的点分成两个集合，使每个集合中的点之间存在的边的长度最小，求这个最小值？

思路 - 并查集

感觉还是只会非常裸的并查集，这种感觉还是想不到如何处理对手关系（只需要构造每个点的对手集合即可）。
先将所有的边按权值降序排序，每次询问时扫一边边，将符合序号的边用并查集处理即可，第一个不满足的边的权值就是所求的最小值。

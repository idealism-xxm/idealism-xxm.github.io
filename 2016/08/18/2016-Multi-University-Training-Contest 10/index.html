1001. Median

题目链接

题目大意

给定一个长度为\(n\)的有序序列，每次选取\([l_1,r_1],[l_2,r_2]\)共\(r_1-l_1+1+r_2-l_2+1\)个数字，求这些数的中位数？

思路 - 可持久化线段树

昨天刚学了区间第\(K\)大，今天就应用上了。
由于可持久化线段树可以加减，所以在模版的基础上直接再加上一个区间即可，中位数根据总的数字个数找一下即可，注意加法会爆\(int\)，转成\(double\)就能\(AC\)了。

看题解后，才注意到有序序列这个特性，所以直接分类讨论即可，并不需要可持久化线段树，如果是无序的就应该用可持久化线段树了。

1002. Hard problem

题目链接

题目大意

求阴影部分面积？

思路Ⅰ - 微积分

看到别人很快就\(AC\)就觉得这是一个不需要微积分直接可以计算的，但是没有画辅助线，不停地推了\(1h\)还是差一个方程，就放弃去\(AC\)了\(1001\)这个模版题。

觉得自己无望求出结果后，把题扔给队友微积分去了，自己又开别的题，期间想到一种非常简洁的坐标系构建，但觉得队友会想到就没去打扰，结果\(1h\)后告诉我她的方法求交点太复杂，放弃了…我把我的方法丢给她继续算，终于历尽千辛万苦\(AC\)了。

坐标系构建：画出正方形的两条对角线，已交点为原点，对角线为坐标轴建立平面直角坐标系，然后求第一象限的\(\frac {1} {4}\)个阴影面积即可。
解出来大概是：\(\frac {r^2( arcsin(\frac {2x} {r}) + \frac {1} {2} sin( 2arcsin(\frac {2x} {r}) ) ) } {8} - \frac {r^2( arcsin(\frac {x} {r}) + \frac {1} {2} sin( 2arcsin(\frac {x} {r}) ) ) } {2} + \frac {\sqrt(2)} {2} rx\)

交点为：\((\frac {\sqrt {14} r} {8} , \frac {\sqrt {2} r} {8})\)
然后将横坐标带入即可。

思路Ⅱ - 平面几何

赛后看到网上这题的解法，发现辅助线大法好，缺条辅助线，什么都不会。。。
具体解法
扇形的圆心角可以通过余弦定理求得：
\( θ_1=arccos(\frac {c^2+d^2-b^2} {2cd} ) \)
\( θ_2=arccos(\frac {b^2+d^2-c^2} {2bd} ) \)
大扇形面积为：\(s_1=\frac {θ_1} {2π} * π r^2 = \frac {1} {2} θ_1 r^2\)
小扇形面积为：\(s_2=\frac {1} {2} θ_2 b^2\)
三角形面积为：\(s_3=\frac {1} {2} cd*sin(θ_1) \)
则阴影面积为：\( 4*(s_2-s_1+s_3) =4*(\frac {1} {2} θ_2 b^2 - \frac {1} {2} θ_1 r^2 + \frac {1} {2} cd*sin(θ_1) \)

1004. Death Sequence

题目链接

题目大意

一排\(n\)个人，编号分别为\(1\~n\)，每次从头开始数的第\(1,k+1,2k+1…\)出局，然后继续从头开始，直至所有人出局，求第\(x\)个出局的人的编号？

思路 - 递推

对于每个人维护两个值，\(rou[i],indx[i]\)表示这个人是第\(rou[i]\)轮的第\(indx[i]\)出局的，再对每一轮维护一个\(cnt[j]\)，表示前\(j\)共出局的人数。为了计算方便零编号为\(0\~n\)，则\(i%k==0\)时，他是第一轮出局的第\(indx[i-k]+1\)个人；若\(i%k!=0\)，则他下一轮的编号为\(i-(\frac {i} {k} +1)\)，即他和本轮编号为\(i-(\frac {i} {k} +1)\)的人，在轮内出局次序是一样的。处理完\(rou,indx\)数组后，再计算每个人的出局次序即可。

1005. Road

题目链接

题目大意

在一条路上有\(n\)个村庄，相邻村庄之间有一段路，共\(n-1\)段，每段路仅能开启、关闭各一次，开启第\(i\)段路一天需要花费\(w[i]\)元。接下来\(m\)天，第\(i\)天会从\(a[i]\)向\(b[i]\)运输货物，求每一天的最小花费？

思路 - 线段树&&树状数组

这个题很简单，但是比赛时以为\(vector\)的\(erase\)是\(O(log(n))\)的，就放心用了，结果一直\(TLE\)，快结束是才发现是\(O(n)\)的…

标程感觉好神奇、好简洁，可就是不明白…
自己用线段树维护每段路的起始天和终止天，然后用树状数组的区间更新单点查询维护每一天的花费，感觉常数有点大。

1006. Counting Intersections

题目链接

题目大意

给定\(n\)条线段，每条线段平行于\(x\)轴或者平行于\(y\)轴，任意两条线段不公用端点且不相互重叠，求有多少个交点？

思路 - 树状数组

做法很明显，由于只有两种类型的线段，且线段之间不共用端点和重叠，所以交点值可能在横线和竖线间产生，将横线和竖线分开处理即可。
将横线拆成起点和终点，起点时对\(y\)值\(+1\)，终点后对\(y\)值\(-1\)，用树状数组维护即可，按照\(x\)升序排序，\(x\)相同时，横线在前，然后扫一边统计即可。

题目大意

709 - B - Checkpoints

题目大意

在数轴上有\(n\)个检查点，第\(i\)的检查点的坐标为\(x[i]\)，有一个人初始在坐标为\(a\)的位置，求这个人到达至少\(n-1\)个检查点走过的最少距离？

思路 - 贪心

根据题意得至多有\(1\)个检查点不会走到，所以可以将所有检查点按坐标升序排序后，不会走到的点一定是最左边的点或者最右边的点，且有方向，所以共有\(4\)种走法，答案为其中最小值。
注意：需要特判n=1时的情况，因为此时可以呆在原地

708 - A - Letters Cyclic Shift

题目大意

给定一个字符串，必须将一段非空子串中的每一个字母单独循环左移，求能获得的字典序最小的字符串？

思路 - 贪心

第一眼看到还以为是\(DP\)，和以前做过的题有点类似，但仔细一想，本题具有优先级，即改小时，修改越前面的越好，改大时，修改越后面的越好，所以就能想到贪心，只改变第一个非\(a\)起的到下一个\(a\)之前的这一段字符串，如果不存在则将最后一个字符变为\(z\)。

708 - B - Recover the String

题目大意

构造一个\(01串\)，使得这个串满足子序列\(00,01,10,11\)的个数为\(a_{00},a_{01},a_{10},a_{11}\)，如果不存在则输出\(Impossible\)。

思路 - 构造

首先可以根据\(a_{00}和a_{11}\)通过\(C_n ^2 = a\)算出\(0和1\)的个数，如过不存在整数解，则无解，否则判断\(a_{01}+a_{10}==n_0*n_1\)是否成立，不成立则无解（因为对于每一个数字\(1\)，其左边的\(0\)的个数+其右边的\(0\)的个数等于\(n_0\)），然后构造出\(a_01\)个\(01\)子序列就好了，因为经过前面的判断，此时必定有解，且满足一个条件就能满足另一个。

计算最左边的\(0\)的个数为\(l_0=\frac {a_{01} } {n_1}\)，则还剩下\(a _{ 01 } \% n_ 1\)个\(01\)序列需要构造，此时只需要\(1\)个\(0\)，让\(a_{01}%n_1\)个在这个\(0\)的右边，其余的在其左边即可完成构造。

708 - C - Centroids

题目大意

给定一棵树，判断每个点是否能至多移动一条边后使其成为重心？

思路 - 树形DP

看了题解后很快就知道如何写了，结果按照自己的方法更新\(mx\)数组，导致最后一直\(WA\)到第二天。。。

令\(cnt[u]\)表示以\(u\)为根的子树的大小，\(mx[u]\)表示\(u\)的子树中能切下的最大的子树大小。
然后第一遍\(dfs\)更新\(cnt[]\)和\(mx[]\)，第二遍\(dfs\)就可以根据求得的结果计算\(ans[]\)。若一个点的某一个子树大于\(\frac {n} {2}\)并且该子树切去他能切下的最大子树后仍大于\(\frac {n} {2}\)，则答案为\(0\)，否则答案为\(1\)。

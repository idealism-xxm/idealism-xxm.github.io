51NOD-1154 回文串划分（DP）

2016-06-13 • 分类: ACM, DP, 区间DP • 标签: 51NOD, ACM, 区间DP

题目链接

1405 树的距离之和

\[基准时间限制：1 秒\ \ 空间限制：131072 KB\ \ 分值: 40\ \ 难度：4级算法题\]

Description

\(有一个字符串S，求S最少可以被划分为多少个回文串。\)
\(例如：abbaabaa，有多种划分方式。\)

\(a|bb|aabaa - 3 个回文串\)
\(a|bb|a|aba|a - 5 个回文串\)
\(a|b|b|a|a|b|a|a - 8 个回文串\)

\(其中第1种划分方式的划分数量最少。\)

Sample Input

1 2
3 2
4 2

Sample Output

5
3
5
5

思路

\(设dp[i]表示前i个字符回文划分的最小值\)
则可得转移方程：若子串\(s[j]…s[i]\)是回文串，则\(dp[i]=min(dp[i],dp[j-1]+1)\)
①很容易就能想到直接判断\(s[j]…s[i]\)是否为回文串，复杂度为\(O(n)\)，则总复杂度为：\(O(n^3)\)
②也可以用\(O(n^2)的时间预处理出以i开始长度为j的子串是否为回文子串\)，然后\(O(1)\)判断，则总复杂度为：\(O(n^2)\)

但是代码交上去，跑的时间很奇怪，\(O(n^2)的跑了800ms多，O(n^3)的才跑了200ms\)不到
